# CS 340 Portfolio – Grazioso Salvare Dashboard

## Project Overview
This project is an interactive dashboard built using Python, Dash, and MongoDB to help Grazioso Salvare identify dogs suitable for different rescue operations. The dashboard allows users to filter animal data and visualize results through charts and a geolocation map.

## Reflection

### How do you write programs that are maintainable, readable, and adaptable?

I write maintainable and readable programs by organizing code into clear sections, using meaningful names, and separating responsibilities. In this project, I used a CRUD Python module to handle all database operations separately from the dashboard. This made the dashboard code cleaner and easier to understand.

The advantage of this approach is that changes to the database logic can be made in one place without affecting the rest of the application. In the future, I can reuse this CRUD module in other projects that require database connections, such as dashboards or data-driven applications.

---

### How do you approach a problem as a computer scientist?

I approach problems by first understanding the requirements, then breaking them into smaller steps. For this project, I identified the filtering requirements, database structure, and dashboard features before building each part step by step.

This project was different because it required connecting multiple components, including a database, backend logic, and a user interface. In the future, I would continue using this structured approach by designing systems first, then implementing and testing each part.
